Add a widget with Picasa photos stocks in tags

== Description ==

The widget allows to add your images based on tags being able to form the size of the thumbnail, the number of images to showing, the use of lightbox to show the image or in new page.

== Installation ==
1. Go to http://oscar-fernandez.es/blog/picasa-tag-widget/
2. Download the zipped plugin file to your local machine
3. Unzip the file
4. Upload the 'picasa-tag-widget' folder to the '/wp-content/plugins/' directory
5. Activate the plugin through the 'Plugins' menu in WordPress
6. Go to the Presentation menu, and click 'Sidebar Widgets' (You will need the Sidebar Widget plugin installed)
7. Drag "Picasa_Tag" to your Sidebar, and click the icon next to it to show the configuration details.
